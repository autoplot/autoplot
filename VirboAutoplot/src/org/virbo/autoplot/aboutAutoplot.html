Autoplot Version v2011a

Autoplot is a browser for the data on the web. Data from various formats can be displayed as they are adapted to a common data model.

Autoplot uses das2, an open-source java library for data visualization and data analysis created by the Radio and Plasma Wave Group at the University of Iowa.

Autoplot was developed primarily under NASA Grant 05-S3CVO05-11 ("Virtual Observatories") and under various other grants.
